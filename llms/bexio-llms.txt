# bexio API

> bexio is a Swiss cloud business-management platform for SMBs and self-employed, covering accounting, invoicing (incl. Swiss QR-bill), contacts, sales orders, projects and time tracking, banking, purchasing, and payroll. The REST API at api.bexio.com/2.0 exposes these modules over HTTPS with JSON, secured by OAuth 2.0 (OpenID Connect) or Personal Access Tokens.

This llms.txt was GENERATED by API Evangelist from the bexio catalog entry and repository artifacts (bexio publishes no first-party llms.txt as of 2026-07-17).

## APIs

- [bexio Contacts API](https://docs.bexio.com/): contacts, relations, groups, sectors (scopes contact_show/edit)
- [bexio Sales Order Management API](https://docs.bexio.com/): quotes, orders, invoices, Swiss QR-bill PDFs (scopes kb_offer_*, kb_order_*, kb_invoice_*)
- [bexio Accounting API](https://docs.bexio.com/): chart of accounts, VAT/MWST, currencies, manual journal entries (scope accounting)
- [bexio Items API](https://docs.bexio.com/): items/products with sale/purchase prices and stock (scopes article_show/edit)
- [bexio Projects & Time Tracking API](https://docs.bexio.com/): projects, timesheets, tasks (scopes project_*, task_*)
- [bexio Files API](https://docs.bexio.com/): file and attachment upload/management (scope file)
- [bexio Purchase API](https://docs.bexio.com/): supplier bills and expenses (scopes kb_bill_*, kb_expense_*)
- [bexio Payroll API](https://docs.bexio.com/): payroll employees, absences, paystubs (scopes payroll_*)
- [bexio Banking API](https://docs.bexio.com/): bank accounts and payments, IBAN/QR-bill reconciliation (scopes bank_account_*, bank_payment_*)
